Что интересного обо мне? Следующее:

Мой путь во frontend начался в апреле 2022 г. В июне выпал шанс без затрат пройти курс профессиональной переподготовки “JavaScript - программирование” с 0-ля на платформе Netology (с июля по май 2023 г.). После курса занялся своими пет - проектами, в процессе которого обрел немало новых навыков как в верстке, так и в механизме работы веб - приложений.

Неустанный ученик, вечно ищу что можно узнать нового или как еще можно улучшить свои навыки. Легко ловлю концепцию дизайна и логически могу построить план веб - проекта. Скрупулезен к своему делу, но понимаю когда стоит остановиться - “Лучшее - враг хорошего”.

Хорошо адаптируюсь в коллективе, не конфликтный и спокойный человек. Готов подолгу обсуждать проект и думать над задачами. Ответственно подхожу к решению поставленных мне задач, хорошо сделанная работа - довольный клиент, следовательно рост компании и в том числе моя репутация.

Не мало важная информация о моем здоровье - есть проблемы со здоровьем, вследствии чего периодически требуются обследования. Поэтому желательно работа из дома (удаленно) или поиск компромиссов.

Кратко, о чем следует дополнить:

  • 1996 года рождения, Р-н-Д.
  • Высшее, бакалавриат “Автоматизация, мехатроника и управление, Автоматизация технологических процессов и производств” (ДГТУ, Р-н-Д, 2018 г);
  • Высшее, магистратура “Технология машиностроения, Конструкторско-технологическое обеспечение машиностроительных производств” (ДГТУ, Р-н-Д, 2020 г);
  • Курс "JavaScript-программирование" (Netology).
  • Знание английского языка на уровне ~А2/B1.
  • Есть собственный автомобиль.

Проекты

Некоторые из заслуживающих внимания проектов, которые я создал:

Wawe

Лендинг вымышленной школы серфинга. Дизайн макет от неизвестного автора. Верстался с веб - сервиса Figma, адаптирован под разные устройства. Небольшой код на PHP для выдачи изображений в галерее.

  • HTML
  • CSS
  • JavaScript
  • ModX
  • Figma
  • PHP
  • API

Noemi

Изначально этот макет был дан на курсе от Netology, где по заданию нужно было только сверстать и сделать адаптивным под разные устройства, т.е. без JS языка. И уже после обучения самостоятельно был дописан недостающий JS код, что “оживил” сайт и сделал его вполне полноценным и самодостаточным. К сожалению кроме главной страницы других страниц.

  • HTML
  • CSS
  • JavaScript
  • ModX
  • Photopea

"Идем В Кино"

Дипломный проект от Netology, где главной задачей было дописать JS код и в частности взаимодействие с API сервером (не моим) по выдаче кинопремьер, заполненность кинозалов и бронирование билетов с использованием стороннего скрипта создания QR - кода.

  • JavaScript
  • ModX
  • API
